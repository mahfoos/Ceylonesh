    Software engineering undergraduate

    The web assignment placed on me required that I create a Main pagebr
    enabling the user to access several parts of the site as
    Task 1. The links should look like buttons and have a hover effect created through the use of CSS.

    Task 2 required creation of a form to send comments about the website.
    It was required that the form include:
    Input fields for users to enter their details (name, email address etc.)
    A field to enter comments
    A way to enable users to enter a rating through the website
    A JavaScript validation to check that the user has filled-in the compulsory
    fields - 'name' and 'rating', without the use of HTML5 validation
    An HTML pop-up window appearing if the name and ratings are filled-in by a user,
    which includes a summary including the name, any comments and rating given on the form
    (Eg:- if the user entered "John" for their name, rated the site as "good" and
    entered "Very informative website", the popup window should say something
    like: "Dear John, Thank you very much for your feedback. You have rated
    our site as Good and your comment was Very informative website.")

    It was also required that I support Consistency of style in the website
    using a unique external style sheet file main page and navigation,
    which could be used throughout the website
