I have played the Development role of Student 4 for this Website

I was required to create a sitemap depicting the navigation structure of the Website.

part of the group website, and it was also required to be developed in SVG.

Task 2 required that I create a page which allows the user to view images.
This page was required to contain a form, 5 thumbnail images,
and 2 pull-down menus (to select the page background colour and the page text colour),
along with an area to display a larger image and associated description of that image.
It was also required that a corresponding large image and the description
for that image would display within the defined location on the form,
whenever a thumbnail was selected, by using MouseOver or radio buttons to select the thumbnail.
